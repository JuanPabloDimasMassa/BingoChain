Configurar Red Sepolia para Pruebas

¿Por qué Sepolia? Sepolia es una red de prueba de Ethereum que simula el comportamiento real de la mainnet pero con ETH gratuito. ¡Perfecto para probar tu billetera real sin costos!
1
Agregar Red Sepolia a MetaMask

Agrega la red de test Sepolia a tu MetaMask:

Nombre de Red: Sepolia Test Network
RPC URL: https://sepolia.infura.io/v3/9aa3d95b3bc440fa88ea12eaa4456161
Chain ID: 11155111
Símbolo: ETH
Block Explorer: https://sepolia.etherscan.io
Tip: También puedes ir a MetaMask → Configuración → Redes → Agregar Red y llenar los datos manualmente.
